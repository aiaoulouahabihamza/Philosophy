# 📚 تطبيق مراجعة الفلسفة - 2 باك

تطبيق ويب تقدمي (PWA) شامل لمراجعة مادة الفلسفة لتلاميذ الباكالوريا - شعبة العلوم الإنسانية. يوفر التطبيق واجهة سهلة الاستخدام مع ميزات متقدمة للبحث والعمل بدون إنترنت.

## ✨ الميزات الرئيسية

### 🔍 البحث المتقدم
- **بحث فوري** عن المفاهيم والأطروحات والفلاسفة
- **اختصار لوحة المفاتيح** (Ctrl+K) للبحث السريع
- **ترتيب ذكي** للنتائج حسب الملاءمة
- واجهة بحث سهلة وسريعة

### 📱 تطبيق ويب تقدمي (PWA)
- **التثبيت على الجهاز** كتطبيق محلي
- **العمل بدون إنترنت** مع تخزين مؤقت ذكي
- **تحديثات تلقائية** للمحتوى
- **صفحة بديلة** عند قطع الاتصال

### 🎨 تصميم عصري وجميل
- **ألوان متناسقة** وهادئة (Azure, Gold, Taupe, Sky)
- **خطوط عربية حديثة** (Cairo Font)
- **واجهة مستجيبة** تعمل على جميع الأجهزة
- **تجربة مستخدم سلسة** وممتعة

### 📖 محتوى شامل
- **3 مجزوءات** فلسفية رئيسية
- **مفاهيم وأطروحات** لكل محور
- **أقوال مأثورة** للفلاسفة العظماء

### ⚡ أداء عالي
- **تحميل سريع** للصفحات
- **معالجة قوية** للأخطاء
- **رسائل خطأ واضحة** وودية

## 🚀 البدء السريع

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت (للتحميل الأول فقط)

### التثبيت

#### الطريقة 1: التشغيل المحلي
```bash
# استخدم Python:
python3 -m http.server 8000

# ثم افتح: http://localhost:8000
```

#### الطريقة 2: التثبيت كتطبيق
1. افتح التطبيق في متصفحك
2. انقر على أيقونة التثبيت
3. اختر "تثبيت"

## 📖 طريقة الاستخدام

### التنقل بين الأقسام
1. **الصفحة الرئيسية**: اختر المجزوءة
2. **المفاهيم**: اختر المفهوم
3. **المحاور**: اختر المحور
4. **الدرس الكامل**: اقرأ المحتوى

### البحث
- **اضغط Ctrl+K** أو انقر على أيقونة البحث
- **اكتب كلمة البحث**
- **اختر النتيجة**
